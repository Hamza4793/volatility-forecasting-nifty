# Volatility Forecasting Project

This project compares econometric and machine learning models for forecasting market volatility using NIFTY 50 data.

## Models Used
- ARCH
- GARCH(1,1)
- Random Forest
- LSTM

## Evaluation Metrics
- MSE
- MAE
- RMSE

## Key Finding
GARCH model outperformed machine learning models, indicating strong volatility persistence.

## How to Run
1. Install requirements
2. Run scripts in src/ folder
